# ICT2025_20122005

本リポジトリは、卒業論文で使用したバーチャルキャンパスに関する資料をまとめたものです。
Unityおよびclusterを用いて構築したバーチャル空間、ならびにアンケート設問・分析資料を含みます。

## 含まれる資料
- アンケート設問の全文
- 単純集計・クロス集計結果の図表
- χ²検定を含む分析手順の概要
- 使用ツールおよび分析環境

## ディレクトリ構成
- survey/ : アンケート設問
- figures/ : 図表（論文・要旨使用）
- analysis/ : 分析手順メモ

## 使用環境
- Unity
- Cluster
- Excel
- jamovi
