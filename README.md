# ICT2025_20122005

本リポジトリは、卒業論文で使用したバーチャルキャンパスに関する資料をまとめたものです。
Unityおよびclusterを用いて構築したバーチャル空間、ならびにアンケート設問・分析資料を含みます。

## 含まれる資料
- [アンケート設問の全文](survey.md)
- 使用ツール
- アンケートの際に使用した動画
- 作成したワールド

## 使用したツール
- Unity 2021.3.4f1
- Cluster　v3.69（2026年1月26日)
- Blender（3Dモデル調整）
- Freeasy（インターネット調査サービス）
- Excel（データ整理・集計補助）
- jamovi（統計解析ソフトウェア）
